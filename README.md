Python+SQL Project

📌 Project Overview

E-commerce platforms generate massive amounts of transactional data every day. This project performs an in-depth analysis of the Brazilian E-Commerce (Olist) Dataset using SQL and Python to uncover valuable insights related to customer behavior, sales performance, payment trends, product categories, and seller performance.

The objective of this project is to transform raw e-commerce data into actionable business insights that can support data-driven decision-making and improve overall business performance.

🎯 Objectives
Analyze customer distribution across cities and states.

Identify top-performing product categories.

Examine order and sales trends over time.

Analyze customer purchasing behavior.

Study payment methods and installment preferences.

Evaluate seller performance and revenue contribution.

Measure business growth through cumulative sales analysis.

Generate actionable insights using SQL queries and data visualization.
